#Optimizing Hospitality: Data-Driven Insights into India's Hotel Economy

##Project Overview
This project analyses hotel pricing, ratings and value-for-money trends across 8 Tier-1 Indian cities.

The workflow includes:
- Data collection using a chrome web scraping extension
- Data cleaning using Excel and Python
- Exploratory Data Analysis (EDA) using Python
- Interactive dashboard creation using Power BI

## Tools used
- Python
- Pandas
- Matplotlib
- Excel
- Power BI
- Chrome Web Scraping Extension

## Dataset
- 8 Tier-1 Indian Cities
- ~150 hotels per city
- Combined into merged dataset
- Key fields: Hotel Name, City, Price, Rating, Number of Reviews

## Dashboards
### Dashboard 1 - Cross-City Comparision
- Average price by city
- Average rating by city
- Price vs rating relationship
- City-level KPI comparision

### Dashboard 2 - City-Level Analysis
- Price distribution
- Rating composition
- Price vs rating scatter plot
- Top 5 hotels of selected city

## Key Insights
- Higher price does not consistently lead to higher ratings
- Ratings are concentrated in a narrow band across cities
- Mid-priced hotels often outpereform premium hotels
- Review volume improves rating reliability

## Business Recommendations
- Align pricing with customer-perceived value
- Improve service quality to justify premium pricing
- Highlight value-for-money hotels on travel platforms
- Encourage customer reviews to build trust
